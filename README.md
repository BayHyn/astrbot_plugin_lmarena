## 🤝 介绍

对接lmarena调用nano_banana等模型进行生图，如手办化

## 📦 安装

- 先前往 [LMarenaBridge](https://github.com/Lianues/LMarenaBridge)安装好LMarenaBridge, 确保该服务器在 <http://127.0.0.1:5102> 启动，确保油猴脚本已成功连接到本地服务器。

- 再安装本插件，可以直接在astrbot的插件市场搜索astrbot_plugin_lmarena，点击安装
